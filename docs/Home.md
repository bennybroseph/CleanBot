# CleanBot

**A clean interface to manage Playerbots. No more complex trees of buttons.**

CleanBot is an addon for World of Warcraft **3.3.5a (Wrath of the Lich King)** that lets you
command [mod-playerbots](https://github.com/mod-playerbots/mod-playerbots) from a simple window,
a right-click menu, and an optional floating action bar — instead of memorizing whisper commands.

## Quick start

1. Install the addon into your `Interface/AddOns` folder.
2. Type **`/cb`** (or **`/cleanbot`**), or left-click the wrench on your minimap.
3. *(Optional)* Install the [MultiBot Bridge](bridge-protocol.md) server module for silent,
   instant control. Without it, CleanBot falls back to whispers — everything still works.

New here? Start with **[Getting Started](getting-started.md)**.
